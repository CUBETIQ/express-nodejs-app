# Express Nodejs App

-   Nodejs
-   Express
-   TypeScript
-   Prettier
-   ESLint
-   Docker
-   DroneCI
-   Lerna Support

# Quickstart

-   Start

```shell
yarn start
```

-   Navigate

```shell
curl http://localhost:3000
```

-   Response

```json
{
    "startedAt": "2021-09-13T13:21:04.184Z",
    "message": "Instance id: presenter#143470",
    "status": "OK"
}
```
